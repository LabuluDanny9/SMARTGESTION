# Projet - Structure de base

Structure minimale pour démarrer un nouveau projet (React + Express).

## Structure

```
SmartGestion/
├── backend/          # API Express
│   ├── index.js
│   └── package.json
└── frontend/         # Application React
    ├── public/
    ├── src/
    └── package.json
```

## Démarrage

### Frontend
```bash
cd frontend
npm start
```
Ouvre http://localhost:3000

### Backend
```bash
cd backend
npm start
```
API sur http://localhost:5000
